# Yggdrasil
*C++17 / Vulkan 1.2 Windows-Only Game Engine with a heavy emphasis on high fidelity rendering.*

## Requirements
* CMake 3.18
* Vulkan SDK 1.2.162.1 or higher
* Visual Studio 2019

## Building
* Go to scripts/
* Run `GenerateVS2019.bat`
* Open the `.sln`-file which was generated in `build/`.
* Build the Solution.
